# dotfiles

## Install and deploy

```
$ curl -L dot.basd4g.net | sh

# or
# curl -L http://raw.githubusercontent.com/basd4g/dotfiles/master/bin/install.sh | sh
```

## clone

```sh
git clone https://github.com/basd4g/dotfiles.git ~/dotfiles
cd ~/dotfiles
```

## add new dotfile

```sh
$ mv ~/.hogerc ~/dotfiles/
$ ln -s ~/dotfiles/.hogerc ~/.hogerc
```
